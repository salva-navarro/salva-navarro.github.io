# Salva Navarro — Professional Portfolio

Portfolio profesional estático publicado en GitHub Pages para presentar el posicionamiento técnico de **Salva Navarro** como **Enterprise PKI Architect**.

## Objetivo

Este sitio está diseñado para:
- Comunicar expertise en arquitectura PKI empresarial.
- Mostrar áreas de especialización técnica (ADCS, CLM, gobernanza criptográfica, Zero Trust y post-quantum readiness).
- Centralizar acceso a publicaciones y repositorios técnicos.

## Estructura del repositorio

- `index.html`: página principal (contenido + estilos + SEO metadata).
- `assets/og-image.svg`: imagen social usada para Open Graph/Twitter cards.
- `robots.txt`: directivas para crawlers.
- `sitemap.xml`: sitemap para indexación.
- `googlead6d967eeb046fd4.html`: verificación de Google Search Console.

## Edición rápida

1. Modifica el contenido textual del portfolio en `index.html`.
2. Si cambias posicionamiento o narrativa, actualiza también:
   - `<title>`
   - `meta description`
   - etiquetas Open Graph y Twitter
   - JSON-LD (`Person`)
3. Si cambias imagen social, reemplaza `assets/og-image.svg` y conserva la URL canónica.

## Checklist antes de publicar

- [ ] Validar que el contenido refleja el enfoque actual del portfolio.
- [ ] Revisar enlaces externos (GitHub profile y repositorios).
- [ ] Comprobar `canonical` y metadatos OG/Twitter.
- [ ] Confirmar que `robots.txt` y `sitemap.xml` apuntan al dominio correcto.
- [ ] Revisar visual en escritorio y móvil.
